Section 1 — Policy Summaries

Encryption Policy

MediAxis encrypts all data in transit using TLS 1.2+ and all stored data using industry‑standard encryption provided by our cloud infrastructure. No PHI is transmitted or stored without encryption.

Access Control Policy

Access to the MediAxis platform is role‑based. Clinic staff can only view or modify information appropriate to their assigned role. Administrative functions are restricted to authorized clinic administrators.

Authentication & MFA Policy

All users authenticate using secure, password‑based login. Multi‑factor authentication (MFA) is supported and will be required for administrative users as the platform expands.

Audit Logging Policy

MediAxis records key claim‑related actions, including creation, updates, and status changes. Logs include user ID, timestamp, and action type. Logs are immutable and retained for audit purposes.

Session Timeout Policy

User sessions automatically expire after a period of inactivity to reduce unauthorized access risk. Users must re‑authenticate to continue working.

Data Retention Policy

MediAxis retains claim‑related data only as long as required for operational and regulatory purposes. No PHI is included in this audit packet.
